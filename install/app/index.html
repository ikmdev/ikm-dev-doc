Install Komet

Komet is available for both Windows and Mac. Follow this visually enhanced guide for a smooth installation experience.

📥 Download Installer​

Operating SystemSteps
Windows(1️) Go to the Komet GitHub page and download the latest Windows installer (.msi).
(2️) Save the file to your chosen location.
MacOS(1) Visit the Komet GitHub page and download the latest Mac installer (.pkg) for your machine (M1 or Intel).
(2️) Save the file to your desired location.

🔧 Run Installer​

Operating SystemSteps
Windows(1) Open the Komet Installer file.
(2) Click "Run" when prompted.
(3) Verify the install by searching for Komet in the Start menu.
MacOS(1) Open the Komet installer file.
(2) Click "Run" when prompted.
(3) Check the installation by using Spotlight Search or Finder.

🚀 How to Run Komet​

Komet requires data for full functionality. Here's a step-by-step guide:

  1. Open Komet: Start the Komet application.
  2. Select Data Source: Choose “Open SpinedArrayStore” from the options.
  3. Confirm Selection: Click “OK” to finalize. Komet is now ready!

📦 How to Import Data​

Follow these streamlined steps to import data into Komet:

  1. Prepare Data Location: Move the data zip file to the appropriate directory:
    • Windows: OSDisk (C:) -> Users -> [your username] -> Solor
    • Mac: Users > [your username] > Solor
    Note: Create a 'Solor' folder if it doesn't exist.
  2. Extract Data: Unzip and extract the data in the Solor folder.
  3. Restart Komet: Relaunch Komet and set “Open SpinedArrayStore” as the data source.
  4. Choose Data Source: Select the new data source and click “OK”. Komet is now ready!

🛠️ Troubleshoot​

For detailed instructions or troubleshooting, refer to the Komet User Guide or submit an issue on GitHub.
